# 🌙 Luna Business AI

O **Luna Business AI** é um painel de inteligência artificial estratégico desenvolvido para otimização e análise inteligente de negócios. O sistema conecta-se a um banco de dados na nuvem para armazenar informações e utiliza modelos de linguagem avançados para gerar insights automáticos e gráficos interativos.

## 🚀 Funcionalidades
- **Painel de Controlo Interativo:** Visualização de dados e métricas em tempo real através de gráficos dinâmicos.
- **Inteligência Artificial Integrada:** Consultas inteligentes e geração de insights de negócios através da API da Groq.
- **Persistência de Dados:** Integração total com base de dados PostgreSQL (Neon.tech) para armazenamento seguro.
- **Interface Web Moderna:** Desenvolvido integralmente com Streamlit para uma experiência de utilizador fluida.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.12
- **Framework Web:** Streamlit
- **Inteligência Artificial:** Groq API
- **Base de Dados:** PostgreSQL (Neon.tech)
- **ORM / Conexão:** SQLAlchemy / Psycopg2
